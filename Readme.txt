1. The Folder contains jar file which starts the application.
2. You can run the application as: 
	Go to jar folder ( where the jar file is located.) (e.g. cd Desktop/)
		java -jar davisbase.jar
3. It will open a prompt davisql>
4. Commands and their supported syntaxes can be referred from the attached pdf.
5. Update and delete are supported only on integer primary key fileds.
6. The Source files are in src folder.
7. After starting the application, data directory will get created at the current location.
8. data/catalog directory store the meta data information.
9. data/user_data stores the user table information.
